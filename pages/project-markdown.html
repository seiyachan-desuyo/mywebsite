Markdown 转换器 Markdown Converter Markdown 変換ツール

一款轻量、优雅的网页端 Markdown 转换工具，极大优化创作者排版工作流。 A lightweight and elegant web-based Markdown conversion tool, streamlining the formatting workflow for creators. 軽量でエレガントなWebベースのMarkdown変換ツール。クリエイターのワークフローを効率化します。

项目背景 Background 背景

我一直习惯使用 Markdown 来记录课程笔记，但我发现将 Markdown 转换为其他格式的文件（如 PDF、Word 等）总是充满麻烦。而且，对于不习惯使用 VSCode 等专业开发者应用的普通用户来说，去配置环境来预览和转换 Markdown 也有着很高的门槛。因此，我利用两周的时间开发了这个纯网页端的轻量级工具。 I have always been used to taking course notes in Markdown, but I found it quite troublesome to convert Markdown files into other formats like PDF or Word. Moreover, for regular users who are not accustomed to professional developer tools like VSCode, setting up an environment to preview and convert Markdown is a high barrier. Therefore, I spent two weeks developing this pure web-based lightweight tool. 私はいつもMarkdownで講義ノートを取っていますが、MarkdownをPDFやWordなどの他の形式に変換するのは非常に面倒だと気付きました。さらに、VSCodeのようなプロ向けの開発ツールに慣れていない一般のユーザーにとって、Markdownをプレビューしたり変換したりする環境を設定するのはハードルが高いです。そこで、2週間かけてこの純粋なWebベースの軽量ツールを開発しました。

💡 核心亮点： 支持开启可视化编辑（富文本模式）。用户可以直接在可视化的右侧界面中修改文字和排版，而无需在左侧繁琐的代码堆里寻找修改点！ 💡 Core Highlight: Supports Visual Editing (Rich Text Mode). Users can directly modify text and formatting in the visual interface on the right side, without having to dig through tedious code on the left! 💡 主要な特徴： ビジュアル編集（リッチテキストモード）をサポート。ユーザーは左側の煩雑なコードの中を探し回ることなく、右側のビジュアルインターフェースで直接テキストや書式を修正できます！

主要特性 Key Features 主な機能

  • 双栏分屏，实时预览： 左侧编写 Markdown 代码，右侧秒级同步渲染对应的富文本效果。 Split-screen Real-time Preview: Write Markdown code on the left, and instantly see the synchronized rich-text rendering on the right. 分割画面のリアルタイムプレビュー： 左側でMarkdownを書き、右側で同期されたリッチテキストのレンダリングを即座に確認できます。
  • 一键复制： 内置复制富文本功能，完美兼容微信公众号、知乎、各大博客平台的网页编辑器。 One-click Copy: Built-in rich text copy function, perfectly compatible with web editors of WeChat, Zhihu, and major blog platforms. ワンクリックコピー： リッチテキストのコピー機能が組み込まれており、主要なブログプラットフォームのWebエディタと完全に互換性があります。
  • 定制化样式与代码高亮： 预设了优雅的排版 CSS，并引入了程序员友好的代码块语法高亮。 Custom Styles & Code Highlighting: Pre-set with elegant typography CSS and developer-friendly code block syntax highlighting. カスタムスタイルとコードハイライト： エレガントなタイポグラフィCSSと、開発者に優しいコードブロックの構文ハイライトが事前設定されています。
