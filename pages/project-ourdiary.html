OurDiary电子手帐 OurDiary Digital Journal OurDiary 電子手帳

一款采用 Vue3 + Node.js 开发的全栈手帐本应用，支持 3D 翻页、Fabric.js 自由画板及私密房间同步。 A full-stack journal app built with Vue3 & Node.js, featuring 3D page flips, Fabric.js freeform canvas, and private room sync. Vue3とNode.jsで構築されたフルスタックの手帳アプリ。3Dページめくり、Fabric.jsの自由なキャンバス、プライベートルームの同期機能を備えています。

项目简介 Project Overview プロジェクト概要

OurDiary（手帐本 / CozyDiary）是一个轻量级、高颜值的数字手帐本 Web 应用。它提供了高度模拟真实日记本的 3D 翻页交互体验，并支持通过自由画板功能进行手写、贴纸、文字排版等创作。 OurDiary (CozyDiary) is a lightweight, highly aesthetic digital journal web app. It provides a highly realistic 3D page-flipping interaction experience and supports free handwriting, stickers, and text typography through a freeform canvas feature. OurDiary（CozyDiary）は、軽量で美しいデジタル電子手帳Webアプリです。非常にリアルな3Dページめくりインタラクション体験を提供し、自由なキャンバス機能を通じて手書き、ステッカー、タイポグラフィの作成をサポートします。

🚀 核心亮点： 采用前后端分离架构（Vue 3 + Node.js + Prisma），支持创建带密码保护的“房间”（私密手帐本），实现画布状态的无损云端持久化，让你的日常记录安全又充满乐趣！ 🚀 Core Highlight: Uses a decoupled front/back-end architecture (Vue 3 + Node.js + Prisma) supporting password-protected "Rooms" (private journals) and lossless cloud persistence of canvas states, keeping your daily records safe and fun! 🚀 主要な特徴： フロントエンドとバックエンドの分離アーキテクチャ（Vue 3 + Node.js + Prisma）を採用し、パスワード保護された「ルーム」（プライベート手帳）の作成とキャンバス状態の無損失クラウド永続化をサポートし、毎日の記録を安全で楽しいものにします！

主要特性 Key Features 主な機能

  • 📖 沉浸式 3D 翻页与自适应： 基于 page-flip 引擎高度还原纸质书物理翻页，支持鼠标拖拽书角。宽屏呈现双页平铺模式，移动端自动适配单页模式，保证阅读体验。 📖 Immersive 3D Page Flip & Responsive: Highly restores the physical page flipping of paper books using the page-flip engine, supporting mouse dragging on corners. Displays a two-page spread on wide screens and adapts to single-page on mobile. 📖 没入型 3D ページめくりとレスポンシブ対応： page-flip エンジンを使用して紙の本の物理的なページめくりを高度に再現し、角のマウスドラッグをサポートします。ワイド画面では見開き2ページを表示し、モバイルでは単一ページに適応します。
  • 🎨 强大自由的排版画板： 基于 Fabric.js 提供类似专业编辑器的创作能力。支持自由手写涂鸦、插入富文本并自定义样式、上传图片素材并拖拽缩放拼贴，甚至可以叠加贴纸。提供完善的图层与撤销/重做管理。 🎨 Powerful Freeform Canvas: Built on Fabric.js, providing pro-editor-like creative capabilities. Supports free handwriting, inserting rich text, uploading/collaging image assets, adding stickers, and full layer & undo/redo management. 🎨 強力な自由タイポグラフィ・キャンバス： Fabric.js を基盤とし、プロのエディタのような創作機能を提供します。手書きの落書き、リッチテキストの挿入とスタイル設定、画像素材のアップロードやドラッグによるコラージュ、ステッカーの重ね合わせに対応。完全なレイヤー管理と取り消し/やり直し管理も備えています。
  • 🔒 私密房间与云端同步： 每个手帐本都在一个独立房间中并支持密码保护。所有画板上的笔画、图片位置与样式（JSON/SVG）都能通过 Node.js + SQLite/Prisma 安全无损地持久化到云端，跨设备无缝同步。 🔒 Private Rooms & Cloud Sync: Each journal is in an independent room with password protection. All strokes, image positions, and styles (JSON/SVG) are safely persisted to the cloud via Node.js + Prisma, syncing seamlessly across devices. 🔒 プライベートルームとクラウド同期： 各手帳は独立した部屋にあり、パスワード保護をサポートします。キャンバス上のすべてのストローク、画像の位置、スタイル（JSON/SVG）は、Node.js + Prisma を介してクラウドに安全に永続化され、デバイス間でシームレスに同期されます。
  • 🖨️ 导出与存档： 集成了 html2canvas 和 jsPDF，支持一键将当前单页渲染导出为高清 PNG/JPG，或者将整个手帐本所有页面打包成一个完整的 PDF 文档，用于永久离线存档。 🖨️ Export & Archive: Integrated with html2canvas and jsPDF, allowing one-click export of the current page as high-res PNG/JPG, or bundling the entire journal into a complete PDF document for permanent offline archiving. 🖨️ エクスポートとアーカイブ： html2canvas と jsPDF を統合し、現在の単一ページを高解像度の PNG/JPG としてワンクリックでエクスポートしたり、手帳全体の全ページを1つの完全な PDF ドキュメントにまとめて永久的なオフラインアーカイブにしたりできます。
