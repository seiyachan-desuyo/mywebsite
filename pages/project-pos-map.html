POS Map POS Map POS Map

把门店主数据、商圈定义和区县边界整合到一张可交互地图中，用于快速查看北京 POS 网络分布。 An interactive map that combines store master data, trade area definitions, and district boundaries to explore the Beijing POS network at a glance. 店舗主データ、商圏定義、行政区境界を 1 枚のインタラクティブ地図に統合し、北京 POS ネットワークを俯瞰できるプロジェクトです。

一、数据整合与主表生成 (Data Merge Pipeline) 1. Data Merge Pipeline 一、データ統合パイプライン

这个项目先把多份 Excel 门店数据整理成统一主表，再基于主表生成可直接在浏览器打开的地图页面。数据处理脚本会合并位置表、P4Q 表和 CQ 表，清洗门店 ID、过滤活跃门店、补充经纬度与店型字段，并输出一份后续地图渲染可以直接使用的主数据文件。 The project first consolidates multiple Excel sources into a single master table, then uses that table to generate a browser-ready map page. The merge script combines location, P4Q, and CQ sheets, cleans store IDs, filters active stores, appends coordinates and POS type fields, and produces a clean dataset for map rendering. このプロジェクトでは、まず複数の Excel 店舗データを 1 つの主テーブルに統合し、その主テーブルをもとにブラウザで直接開ける地図ページを生成します。統合スクリプトは位置表、P4Q、CQ を結合し、店舗 ID を整理し、稼働店舗を抽出し、座標と店型情報を補完して、地図描画用のベースデータを出力します。

二、交互式地图展示 (Interactive Store Map) 2. Interactive Store Map 二、インタラクティブ地図表示

地图端基于 Leaflet 生成，集成了门店原点、门店辐射范围、商圈多边形和北京区县边界等图层，并提供图层控制、门店类型筛选、ID 或名称搜索、搜索候选列表和图例卡片。点击商圈时会高亮关联门店，点击门店时也能反向定位对应商圈，便于快速理解覆盖关系和网络分布。 The map interface is built with Leaflet and combines store origin points, store coverage circles, trade area polygons, and Beijing district boundaries. It includes layer toggles, POS type filtering, ID or name search, candidate result lists, and legend cards. Clicking a trade area highlights linked stores, while clicking a store can reverse-focus the related trade area to make network coverage easier to read. 地図画面は Leaflet をベースに構築され、店舗ポイント、カバレッジ円、商圏ポリゴン、北京の行政区境界を統合しています。レイヤー切替、店型フィルター、ID または名称検索、候補リスト、図例カードも備えており、商圏クリック時の関連店舗ハイライトや、店舗クリック時の商圏逆フォーカスによって、分布とカバー関係を素早く把握できます。

💡 项目重点：核心不是单纯“画一张地图”，而是把门店主数据清洗、商圈表达、范围规则和检索交互放到一个轻量可维护的 HTML 产物里，方便后续直接更新与共享。 💡 Project Focus: The value is not just “drawing a map”, but packaging data cleaning, trade area visualization, coverage logic, and search interaction into a lightweight HTML artifact that is easy to refresh and share. 💡 プロジェクトのポイント：単に地図を描くのではなく、店舗データ整形、商圏表現、カバレッジロジック、検索インタラクションを、更新しやすく共有しやすい軽量 HTML 成果物としてまとめた点にあります。
