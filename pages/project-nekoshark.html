猫猫鲨桌宠 CatShark Desktop Pet 猫猫鮫 デスクトップペット

一款基于 Python 和 PyQt6 开发的趣味桌面虚拟宠物应用。包含 AI 聊天、多款打工小游戏、宠物养成及房贷系统。 A fun desktop virtual pet application developed with Python & PyQt6. Features AI chat, multiple mini-games, pet nurturing, and a mortgage system. PythonとPyQt6で開発された楽しいデスクトップ仮想ペットアプリ。AIチャット、複数のミニゲーム、ペット育成、住宅ローンシステムを備えています。

项目简介 Project Overview プロジェクト概要

在日常的桌面工作中，我希望有一只可爱又带点调皮的虚拟宠物陪伴左右。因此，我开发了这款名为“猫猫鲨 (CatShark)”的桌面宠物模拟器。它不仅拥有完整的生命周期系统，还自带一套极其有趣的虚拟经济和房贷玩法！ During daily desktop work, I wanted a cute and slightly mischievous virtual pet for company. So, I developed this desktop pet simulator called "CatShark". It not only has a complete life cycle system but also features an extremely fun virtual economy and mortgage gameplay! 毎日のデスクトップ作業中、かわいくて少しいたずら好きな仮想ペットの仲間が欲しいと思いました。そこで、「猫猫鮫（CatShark）」というデスクトップペットシミュレーターを開発しました。完全なライフサイクルシステムを持つだけでなく、非常に楽しい仮想経済と住宅ローンゲームも備えています！

🤖 核心亮点： 集成 DeepSeek API 打造沉浸式的智能对话体验，并通过丰富的小游戏（扫雷、数独、俄罗斯方块、五子棋等）赚取金币来维持宠物生活与还房贷！ 🤖 Core Highlight: Integrated DeepSeek API for immersive intelligent chat, and various mini-games (Minesweeper, Sudoku, Tetris, Gobang, etc.) to earn coins for pet care and mortgage payments! 🤖 主要な特徴： 没入型のインテリジェントチャット用のDeepSeek APIを統合し、様々なミニゲーム（マインスイーパー、数独、テトリス、五目並べなど）でコインを稼ぎ、ペットの世話や住宅ローンを支払います！

主要特性 Key Features 主な機能

  • 🍼 完整的状态系统与交互： 拥有静止、睡觉、饥饿、生病、哭泣等丰富状态。你可以给宝宝喂食、洗澡、去医院、去健身房。支持鼠标物理拖拽，高空摔下还会扣血和掉好感度！ 🍼 Complete Status & Interaction: Features various states like idle, sleeping, hungry, sick, and crying. You can feed, bathe, take to hospital or gym. Supports mouse dragging physics—dropping from high places reduces health and affection! 🍼 完全なステータスと対話： 待機、睡眠、空腹、病気、泣きなどの様々な状態があります。餌やり、お風呂、病院やジムへ行くことができます。マウスのドラッグ物理をサポートし、高いところから落とすと体力と好感度が下がります！
  • 💬 AI 智能对话系统： 内置气泡聊天界面，气泡智能跟随宠物移动。猫猫鲨拥有自己的独立人设，随时开启趣味陪伴聊天。 💬 AI Chat System: Built-in bubble chat interface that smartly follows the pet's movement. CatShark has its own unique persona for fun companion chats anytime. 💬 AIチャットシステム： ペットの動きにスマートに追従する吹き出しチャットUIを内蔵。猫猫鮫には独自の設定があり、いつでも楽しいおしゃべりができます。
  • 🎮 多样化打工与经济体系： 想买大别墅？没问题，支持按揭贷款！每天通过打工（玩五子棋、扫雷、数独、俄罗斯方块、飞翔猫猫鲨等内置小游戏）赚取金币还房贷。 🎮 Mini-games & Economy: Want a big mansion? No problem, get a mortgage! Earn coins daily to pay it off by working (playing built-in games like Gobang, Minesweeper, Sudoku, Tetris, Flappy CatShark). 🎮 ミニゲームと経済： 大きな豪邸が欲しい？問題ありません、住宅ローンを組みましょう！毎日働いて（五目並べ、マインスイーパー、数独、テトリス、フラッピーなどの内蔵ゲームをプレイ）ローンを返済します。
