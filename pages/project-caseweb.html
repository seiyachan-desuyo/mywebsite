一、案例摘要搜索 (Case Summary Search) 1. Case Summary Search 一、ケースサマリー検索

提供全文智能搜索和多维度的高级时间/属性筛选。支持表格与卡片双视图模式，双击即可复制内容。内建 KPI 仪表板和数据导出功能，同时可以保存用户的筛选预设与搜索历史。 Offers full-text smart search and multi-dimensional advanced filtering by time and attributes. Supports both table and card views, with double-click to copy functionality. Features built-in KPI dashboards, data export, and saves user filter presets and search history. フルテキストのスマート検索と、時間や属性による多次元の高度なフィルタリングを提供します。テーブルとカードの両方のビューをサポートし、ダブルクリックでコンテンツをコピーできます。組み込みのKPIダッシュボード、データエクスポート機能を備え、ユーザーのフィルタープリセットと検索履歴を保存します。

二、成本追踪可视化 (Brief Cost Tracker) 2. Brief Cost Tracker Visualization 二、コスト追跡の視覚化

成本追踪模块提供 8 个核心字段的侧边栏快速筛选与全局搜索，搭配灵活的数据列自定义视图，帮助决策者直观地洞察总投资、平均 EBITDA 及新旧店比例的分布情况。 The Cost Tracker module provides quick sidebar filtering on 8 core fields and global search, along with flexible custom data column views. It helps decision-makers intuitively gain insights into total investment, average EBITDA, and new vs. renovated store ratios. コスト追跡モジュールは、8つのコアフィールドのサイドバークイックフィルターとグローバル検索を提供し、柔軟なカスタムデータ列ビューと組み合わせて、意思決定者が総投資額、平均EBITDA、および新旧店舗の比率の分布を直感的に洞察するのを支援します。
