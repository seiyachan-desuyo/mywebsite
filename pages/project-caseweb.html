案例与成本数据大屏 Case Data Builder データビルダー

融合极速 OCR 与自研模板引擎，为 Apple Finance 打造的智能化成本追踪分析平台。 An intelligent cost tracking platform built for Apple Finance, integrating fast OCR and a custom template engine. 高速OCRと独自のテンプレートエンジンを統合した、Apple Finance向けのインテリジェントなコスト追跡プラットフォーム。

角色Role役割
Full Stack Dev / Analyst

12 Weeks • Present

项目背景 Background 背景

在 Apple Finance 团队中，分析多维度的成本数据及审批各类门店扩展案例需要耗费大量人工去核对 Excel 表格和 PDF 文件。为了将繁重的工作自动化并打通数据孤岛，我设计并开发了这个名为 Brief Cost Tracker（Case Data Builder）的数据应用平台。 Within the Apple Finance team, analyzing multi-dimensional cost data and approving various store expansion cases previously required extensive manual effort to cross-check Excel sheets and PDFs. To automate this heavy workload and break down data silos, I designed and developed this data application platform named Brief Cost Tracker (Case Data Builder). Apple Financeチームでは、多次元のコストデータを分析し、さまざまな出店ケースを承認するために、ExcelシートとPDFを手作業で照合する膨大な労力が必要でした。この重労働を自動化し、データの孤立を解消するために、Brief Cost Tracker（Case Data Builder）というデータアプリケーションプラットフォームを設計および開発しました。

💼 商业价值： 大幅提升跨维度案例检索的效率，实现成本追踪的可视化，并将原本繁琐的英文商业审批报告实现了「一键生成」，极大解放了财务团队的生产力。 💼 Business Value: Significantly improved the efficiency of cross-dimensional case retrieval, realized visual cost tracking, and achieved "one-click generation" of previously tedious English business approval reports, greatly freeing up the finance team's productivity. 💼 ビジネス価値： 多次元ケース検索の効率を大幅に向上させ、コスト追跡を視覚化し、以前は面倒だった英語のビジネス承認レポートの「ワンクリック生成」を実現し、財務チームの生産性を大幅に解放しました。

核心功能展示 Core Features 主な機能の紹介

  • 多维数据看板 (Streamlit)： 采用 Streamlit 构建响应式前端界面，提供全文智能搜索与双视图模式（表格/卡片），能够快速对海量案例和成本数据进行筛选、联动展示与下钻分析。
  • Multi-dimensional Dashboard (Streamlit): Built a responsive frontend interface using Streamlit, featuring full-text smart search and dual view modes (Table/Card) to quickly filter, link, and drill-down into massive cases and cost data.
  • 多次元ダッシュボード (Streamlit)： Streamlitを使用してレスポンシブなフロントエンドインターフェースを構築し、全文スマート検索とデュアルビューモード（テーブル/カード）を備え、大量のケースとコストデータの迅速なフィルタリングとドリルダウン分析を可能にしました。
  • 极速定点 OCR 与智能解析： 基于 pdf2image 与 easyocr 集成了防内存溢出的极速 OCR 技术。通过生成 72 DPI 缩略图速查锁定页面，再行 300 DPI 定点精确解析，一键提取 PDF 与 Excel 中的关键商业及财务指标。
  • Targeted OCR & Smart Parsing: Integrated high-speed OCR using pdf2image and easyocr with OOM prevention. It uses 72 DPI thumbnails for rapid page targeting followed by 300 DPI precision extraction, pulling key business and financial metrics from PDFs and Excel instantly.
  • 高速OCRとスマート解析： メモリ不足を防ぐために pdf2image と easyocr を使用した高速OCRを統合しました。72 DPIのサムネイルでターゲットページを素早く見つけ、300 DPIの高精度抽出を行い、PDFやExcelから重要なビジネス指標と財務指標を瞬時に抽出します。
  • 自动化文案组装与推荐 (Template Engine)： 搭载自研的模板引擎，自动汇集解析后的数据填入标准化句式中。同时匹配历史数据库，依据多维度权重打分 (Scoring)，智能推荐历史最佳的审批金句供直接采纳。
  • Automated Report Assembly & Recommendation: Features a custom template engine that aggregates parsed data into standardized sentences. It also matches historical databases, scoring them across multiple dimensions to recommend the best historical approval phrases for immediate use.
  • 自動レポート作成と推奨： 解析されたデータを標準化された文に集約するカスタムテンプレートエンジンを搭載しています。また、過去のデータベースと照合し、多次元のスコアリングに基づいて最適な過去の承認フレーズを推奨し、すぐに使用できるようにします。
