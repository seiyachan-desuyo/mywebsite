项目简介 Project Overview プロジェクト概要

RTM Finance Skills Marketplace 是一个面向 Apple RTM Finance 团队的内部技能插件门户。目标是把分散的脚本和自动化能力集中到一个可检索、可分类、可直接复制安装命令的统一入口，降低工具发现成本。 RTM Finance Skills Marketplace is an internal skill portal for Apple RTM Finance. It centralizes distributed scripts and automations into a searchable and categorized entry point with direct install command copy. RTM Finance Skills Marketplace は Apple RTM Finance 向けの社内スキルポータルです。分散したスクリプトと自動化機能を、検索可能で分類済みの単一入口に統合します。

核心价值： 从“知道有工具”到“30 秒内找到并安装工具”。通过中英双语、统一分类和一键复制命令，显著提升团队自动化工具的触达效率。 Core Value: Move from "knowing a tool exists" to "finding and installing it in 30 seconds" via bilingual UX, clear taxonomy, and one-click command copy. コア価値： 「ツールがあると知っている」から「30秒で見つけて導入できる」へ。二言語UXと統一分類で到達性を高めます。

功能设计 Feature Design 機能設計

  • 极速检索： 支持中英双语关键词与模糊匹配，快速定位技能。 Fast Search: Bilingual fuzzy search for quick skill discovery. 高速検索： 二言語のあいまい検索で迅速にスキルを特定。
  • 工作流分类： 按 Mono、Keynote、Data Analyze、Article、Email、Automation 六大类组织。 Workflow Categories: Mono, Keynote, Data Analyze, Article, Email, Automation. ワークフロー分類： Mono、Keynote、Data Analyze、Article、Email、Automation。
  • 一键安装： 自动生成并复制 npx 安装命令，降低上手门槛。 One-click Install: Auto-generate and copy npx installation commands. ワンクリック導入： npx 導入コマンドを自動生成・コピー。
  • 双语界面： 内置 CN/EN 切换，适配跨地区团队协作。 Bilingual UI: Built-in CN/EN switch for global collaboration. 二言語UI： CN/EN 切り替えでグローバル連携に対応。

实现方式 Implementation 実装方式

  • 纯前端架构： 以静态页面方式运行，无需数据库和后端服务。 Static Front-end: Runs as pure static pages without backend dependencies. 静的フロントエンド： バックエンド不要の静的構成。
  • 核心文件： index.html、app.js、style.css 负责 UI、数据渲染与交互逻辑。 Core Files: index.html, app.js, style.css for layout, rendering, and interaction. 主要ファイル： index.html、app.js、style.css が UI とロジックを担当。
  • 扩展机制： 通过 app.js 的 skillsData 数组即可快速新增 Skill 卡片。 Extensibility: Add new skills by extending the skillsData array in app.js. 拡張性： app.js の skillsData 配列に追加するだけで拡張可能。

预览说明： 该项目为公司内部站点，当前不提供公网预览链接。 Preview Note: This is an internal company website and no public live preview is available. プレビュー注記： 社内サイトのため、公開プレビューは提供していません。

如需演示，可在内网环境通过本地静态服务启动并访问。 For demos, run it in the internal environment via a local static server. デモが必要な場合は、社内環境でローカル静的サーバーを起動して確認します。
